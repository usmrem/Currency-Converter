💱 Currency Converter

A currency converter web application built with HTML, CSS and JavaScript.

This application allows users to convert money between different currencies using real-time exchange rates.

🌐 Live Demo

https://usmrem.github.io/Currency-Converter/

🚀 About The Project

This project was created to practice working with APIs and JavaScript.

The application fetches exchange rate data and calculates currency conversions based on the user input.

🛠 Technologies Used
HTML5
CSS3
JavaScript
Currency Exchange API

📂 Features
Convert between different currencies
Real-time exchange rates
User-friendly interface
Dynamic calculation
Responsive layout
